### `npm ci`

2. Запуск dev-сервера

### `npm run dev`

## Реализация 

Опции компоента `Gallery`
- rowMaxHeight - ограничение максимальной высоты строки
- gap - оступ между объектами

Компонент `Thumbnail` используеться для оптимизации вывода изображений.
